## 简介

- 本项目仓库为个人学习数据结构与算法的日常记录

## 如何使用

**目前项目是两大模块**

**1、数据结构**

+ 位于`data_structure`目录下的是数据结构
+ 每种数据结构安排一个`package`，每个`package`下有对应的`Main`方法用于测试

**2、算法**

+ 位于`algorithm`目录下的是算法模块
+ `sorting`包是编码常见的排序算法
